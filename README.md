# Task 4: Firewall Configuration on Windows

## Objective
To configure and test basic firewall rules using Windows Defender Firewall.

## Tool Used
- Windows Defender Firewall

## Steps Performed
1. Opened Windows Firewall
2. Created inbound rule to block port 23 (Telnet)
3. Created inbound rule to allow port 22 (SSH)
4. Verified firewall rules using Advanced Settings

## Outcome
Learned basic firewall configuration and network traffic filtering
